# Summary
Simon Game - Interactive web app written with JavaScript/HTML/CSS programming language to simulate the classic Simon Electronic Memory Game.

<strong><h1>Game Rule</h1></strong>
Each level will have different set(s) of button blink, player must follow the order of how the button blink and click the corresponding button in order. If all buttons were click correctly, player will enter next level and have a more complex set of challenge as the game progresses to higher levels. ðŸ™ƒ Let's play your memory!! ðŸ™ƒ
